# 🎓 Ассистент ДОУ «Кустук»

Цифровой методический ассистент для воспитателей дошкольного образования с интеграцией региональной программы «Кустук» (Республика Саха (Якутия)).

## 🌟 Возможности

- 📅 Календарно-тематическое планирование
- 📊 Педагогическая диагностика (3 раза в год)
- 🌈 Программа «Кустук» (этнокультурный компонент)
- 🤖 ИИ-помощник (GigaChat / DeepSeek)
- 📄 Экспорт в PDF и Word
- 🖼️ Генерация иллюстраций
- 👥 Раздельный вход (воспитатель/методист/админ)
